The gnu.text Package

The gnu.text package supports various utility classes for formatting, parsing, and manipulating text (strings).

It is used by Kawa.

License

Author

Per Bothner <per@bothner.com>

How to get it

The gnu.text package is currently distributed as part of Kawa, though it can be used independent of the rest of Kawa.

Bugs and patches

Send them to per@bothner.com, or to the Kawa mailing list.
